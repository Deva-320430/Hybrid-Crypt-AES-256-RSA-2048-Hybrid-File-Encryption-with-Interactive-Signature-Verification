# Hybrid-Crypt-AES-256-RSA-2048-Hybrid-File-Encryption-with-Interactive-Signature-Verification
Designed and implemented a secure hybrid cryptosystem for file encryption, decryption, and digital signing. Combined AES-256 symmetric encryption with RSA-2048 asymmetric key wrapping and RSA-PSS signature verification to protect confidentiality, integrity, and authenticity.
